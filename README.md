# movie_api

API that serve a movies database and users who can favorite and score the movies.

API documentation: [Documentation](https://movie-api2.herokuapp.com/docs/ "documentation")  (Built with Swagger UI)


## Technologies used:
* Node.js
* Express
* Mongoose
* Passport
* bcrypt
* Json Web Token
* JSDoc
* OpenAPI 3.x
* Swagger

Hosted on Heroku
