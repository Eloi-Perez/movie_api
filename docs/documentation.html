Time Travel Films API

API Endpoints Documentation

Search time travel films by genre, actors or directors.

  • Request
    Method /Endpoint
    Request Body
    Response Body

  • Get a list of all the movies
    GET /movies
    Res: JSON with movies list.

  • Get a list of all featured movies
    GET /movies/featured
    Res: JSON with all featured movies.

  • Get Movie by title
    GET /movies/<title>
    Res: JSON with description, genre, director, image URL, whether itâ€™s featured or not.

  • Get genre by name
    GET /genres/<genre>
    Res: JSON with Genre's info.

  • Get director by name
    GET /directors/<director>
    Res: JSON with Director's info.


  • Get User info and movies list from User
    GET /users/<user>
    Res: JSON with User info and movies list from User.

  • Register user
    POST /users
    Req: JSON {"Username": <"your_user_name">, "Password": <"your_password">, "Email": <"your@email.com">, "BirthDate": <"YYYY-MM-DD">}
    Res: Successful message.

  • Login
    POST /login
    Req: JSON {"Username": <"your_user_name">, "Password": <"your_password">}
    Res: Successful message.

  • Update user
    PUT /users
    Req: JSON {"Username": <"your_user_name">, "Password": <"your_password">, "NewUsername": <"your_new_user_name">, "NewPassword": <"new_password">, "NewEmail": <"new_email">, "BirthDate": <"YYYY-MM-DD">} Only "Username" and "Password" are mandatory for authentication purposes.
    Res: Successful message.

  • Delete user
    DELETE /users
    Req: JSON {"Username": <"your_user_name">, "Password": <"your_password">}
    Res: Successful message.


  • Add movie to user list
    POST /users/<user>/myMovies
    Req: JSON {"Movie": <"movie_title">, "Score": <num/10>, "RelevanceTT": <num/10> "PlanToWatch": <boolean>, "Favorite": <boolean>}
    Res: Successful message.

  • Update movie attributes from user list
    PUT /users/<user>/myMovies
    Req: JSON {"Movie": <"movie_title">, "Score": <num/10>, "RelevanceTT": <num/10>, "PlanToWatch": <boolean>, "Favorite": <boolean>} Only the "Movie" filed is mandatory.
    Res: Successful message.

    • If you want to delete a movie or field in myMovies use PUT sending empty fields in the JSON:
    {"Movie": <"movie_title">, "Score": "", "RelevanceTT": "", "PlanToWatch": false, "Favorite": false}
